==================================================
PRODUCT VISION
AAAgronegócio is a SaaS + marketplace platform for farm sales.

It has two major areas:

1. Public marketplace website
- Similar to OLX for farms
- Buyers can browse without login
- Search, filters, map, listing pages, premium ranking, request actions
- Social links and institutional presence for the company
- Public platform focused on finding and evaluating farms

2. Private SaaS management platform
- Login-protected
- Used by Admin and authorized Sellers
- Admin has full platform control
- Seller has a premium business dashboard with limited scope for managing only their own farm listings and insights
- Used for listing management, approval workflow, metrics, analytics, financial insights, premium tools, and operational control

==================================================
PRIMARY BUSINESS MODEL
==================================================

- Platform owner acts as the broker/intermediary
- Sellers can submit farm listings
- Admin reviews and approves or rejects listings before publication
- Buyers browse listings and can:
  1. request more information
  2. send an initial proposal
  3. schedule a visit
- These actions do NOT create internal chat
- Instead, each action redirects to WhatsApp or email flow with predefined messaging
- Email goes to the platform owner/broker
- WhatsApp can also be used with predefined templates

Premium sellers have:
1. higher ranking in search
2. featured placement in listings
3. more photos/videos
4. access to SaaS dashboard for their own listings
5. AI-assisted description generation
6. AI title improvement
7. AI price suggestion by comparison
8. advanced analytics for their listings

==================================================
USER ROLES
==================================================

1. Admin
- Full SaaS control
- Reviews and approves/rejects listings
- Manages platform website institutional content
- Manages sellers and buyers
- Sees all dashboards, analytics, and financial data
- Can manage homepage banners and platform images
- Does NOT edit seller listing content directly except moderation/admin actions

2. Seller
- Can create and manage own listings
- Can upload media
- Can edit listing fields
- Can view analytics for own listings
- Can access SaaS only if authorized/premium
- Can upgrade features through premium plan
- Can switch profile mode if needed

3. Buyer
- Can browse public marketplace without login
- Needs login to access more sensitive seller information or advanced actions
- Can save favorites, send interest, request proposal flow, request visit flow
- Can switch profile mode if needed
